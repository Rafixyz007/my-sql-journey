## DDL & DML – Managing Database Structure and Data

### What this section covers
This section focuses on creating, modifying, and manipulating data inside tables.

There are two main categories:

- DDL (Data Definition Language)
  → Defines and modifies database structure
- DML (Data Manipulation Language)
  → Works with the data inside tables

---

### Files in this folder
- `ddl.sql` – CREATE, ALTER, DROP
- `dml.sql` – INSERT, UPDATE, DELETE, TRUNCATE

---

### Key Concepts
- Creating tables with constraints
- Modifying table structure
- Inserting new records
- Updating existing records
- Deleting data safely
- Difference between DELETE and TRUNCATE
