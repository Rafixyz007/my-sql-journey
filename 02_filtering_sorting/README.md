## Filtering and Sorting Data – WHERE & ORDER BY

### What this section covers
This section focuses on **narrowing down results** and **ordering them** to make your queries more useful and readable.

Key topics:
- Filtering rows with `WHERE`
- Using comparison operators (`=`, `!=`, `>`, `<`, `>=`, `<=`)
- Combining conditions with `AND` & `OR`
- Sorting data with `ORDER BY` in ascending or descending order

---

### Files in this folder
- `where_queries.sql` – filtering data using WHERE
- `order_by_queries.sql` – sorting data using ORDER BY

---

### Key Concepts
- Filtering rows to get only the data you need
- Combining multiple conditions
- Sorting results for clarity
- ASC vs DESC
